
Percentage Widget For OpenERP 7.0
========================================================
         author : figolliu@gmail.com
         date   : 2014.7.8

--------------------------------------------------------
主要功能:<br />
--------------------------------------------------------
以百分比分的形式显示数字<br />

<pre>

安装方法:
1. download source code；
2. update module list；
3. install

使用方法：
在field标签中，增加widget="percentage"属性即可

效果：
margin: 1.1  -->  110%
margin: 0.2  -->  20%

</pre>
